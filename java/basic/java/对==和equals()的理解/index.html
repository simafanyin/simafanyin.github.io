对==和equals()的理解

发表于 2020-10-12 更新于 2021-08-17 分类于 java ， basic

==和equals的区别

  • == 是关系运算符，equals()是方法，都返回Boolean类型
  • Object的 == 和 equals()都是比较的地址，作用相同

    == 的作用

  • 基本类型，比较值是否相同
  • 引用类型，比较内存地址是否相同
  • 不能比较没有父子关系的两个对象

    equals() 的作用

  • jdk中的类已经重写过，比较的是内容。
  • 自定义的类没有重写equals()，将调用父类，即Object的equals()方法，其使用this == Obj，比较内存地址
