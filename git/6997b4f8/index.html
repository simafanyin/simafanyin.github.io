GitHub 官方表示，从2020/10/01开始，该平台所有新建的源代码库，缺省名称为 main。
此处记录如何将master分支切换到main分支

切换项目分支(从master –> main)

github目前创建项目缺省分支名为main

1.修改本地缺省分支

2.git新建仓库，本地分支由master变为main

3.合并两个分支，将master分支，合并到main中

4.本地add，commit，并将本地main分支推送到远程仓库
